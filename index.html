TCP-IP And OSI Model

OSI-Model :

The OSI (Open Systems Interconnection) model is a conceptual framework that standardizes the functions of a telecommuni-cation into seven layers, allowing for interoperability between different systems and network protocols. Learn-More

TCP-IP :

TCP / IP ( Transmission Control Protocol / Internet Protocol ) is a communi-cation standard that enables application programs and computing devices to exchange messages over a network. It consists of four layers. Learn-More
Difference Between OSI-Model And TCP-IP
OSI-Model TCP-IP
The full form of OSI is Open Systems Interconnection. The full form of TCP/IP is Transmission Control Protocol/ Internet Protocol.
It is a structured model which deals which the functioning of a network. It is a communication protocol that is based on standard protocols and allows the connection of hosts over a network.
In 1984, the OSI model was introduced by the International Organisation of Standardization (ISO). In 1982, the TCP/IP model became the standard language of ARPANET.
It comprises seven layers:
  • Physical
  • Data Link
  • Network
  • Transport
  • session
  • Presentation
  • Application
 It comprises of four layers:
  • Network Interface
  • Internet
  • Transport
  • Application
It follows a vertical approach. It follows a horizontal approach.
An OSI Model is a reference model, based on which a network is created. The TCP/IP is the implementation of the OSI Model.
It is protocol independent. It is protocol dependent.
